Welcome to the README for SLEPIAN_Hotel, a set of routines for the
computation of spherical harmonics, Slepian functions, and transforms.
This package is hosted by CSDMS at
http://csdms.colorado.edu/wiki/Model:SLEPIAN_Bravo


Where do I start?

he site https://github.com/Slepian/Slepian/wiki contains instructions
and an installation program that sets up the folder structures and
automatically downloads required data files. It also contains some
demos.
